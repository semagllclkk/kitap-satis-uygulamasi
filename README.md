# 📚 Kitapevi Online Satış Sistemi

Yazılım Mühendisliği dersi kapsamında verilen bir ödev olan kitap satış uygulaması 
- Nest.js backend + Next.js frontend

## 🎯 Proje Özellikleri

- **Login Sistemi**: Müşteri ve yönetici ayrı giriş
- **E-Ticaret**: Kitap listesi, sepet, satın alma
- **Yönetici Paneli**: CRUD işlemleri, satış istatistikleri
- **Admin Reset Mekanizması**: Test verilerini sil, gerçek verileri getir
- **Dark Cute Pink Tema**: Profesyonel ve göz alıcı tasarım
- **Satış Grafikleri**: Zaman bazlı satış verileri görselleştirmesi

## 📋 Teknik Stack

- **Backend**: Nest.js + TypeScript + SQLite
- **Frontend**: Next.js + React + TypeScript + Tailwind CSS
- **Database**: SQLite
- **Styling**: Tailwind CSS ile Dark Cute Pink tema

## 🚀 Başlangıç

### Backend Setup
```bash
cd backend
npm install
npm run start
```
Backend: http://localhost:3000

### Frontend Setup
```bash
cd frontend
npm install
npm run dev
```
Frontend: http://localhost:3001
